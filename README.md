# 📘 Online Quiz Application

An Online Quiz Application developed using Java, Swing, JDBC, and MySQL.  
It allows users to register, log in, attempt quizzes, and view their scores.

---

## 🚀 Features
- User Registration & Login  
- Multiple Choice Questions (MCQs)  
- Aptitude & Java Quizzes  
- Score Calculation  
- Result Storage in Database  

---

## 🛠 Tech Stack
- Java (Core Java, JDBC)  
- Swing (GUI)  
- MySQL (Database)  
- Eclipse IDE  

---

## 📂 Project Structure

OnlineQuizApplication
└── src
├── model # Data models (User, Question)
├── dao # Database operations (JDBC)
├── ui # Swing UI (Login, Dashboard, Quiz)
└── Main.java # Entry point

---

## ⚙️ Setup
1. Clone the repository  
2. Create MySQL database `quiz_app`  
3. Update DB credentials in `DBConnection.java`  
4. Add MySQL Connector JAR  
5. Run `Main.java`  

---

## ▶️ Usage
- Register a new user  
- Login with credentials  
- Select a quiz  
- Answer questions  
- View final score
